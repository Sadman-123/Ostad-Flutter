## Getting Started

To run this project locally, follow these steps:

1. **Clone the repository:**
   \`\`\`sh
   git clone <repository_url>
   \`\`\`

2. **Navigate to the project directory:**
   \`\`\`sh
   cd <project_directory>
   \`\`\`

3. **Install dependencies:**
   \`\`\`sh
   flutter pub get
   \`\`\`

4. **Run the application:**
   \`\`\`sh
   flutter run
   \`\`\`
